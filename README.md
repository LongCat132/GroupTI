# GroupTI - 小组作业人格测试

一个趣味心理测试，帮助你发现自己在小组作业中的人格类型。

在线访问: https://longcat132.github.io/GroupTI

## 项目简介

GroupTI (Group Team Indicator) 是一个基于 MBTI 风格的小组作业人格测试。通过 15 道选择题，测试用户在小组成员/组长角色下的行为模式，生成 16 种专属人格类型。

## 核心维度

| 维度 | 字母 | 含义 |
|------|------|------|
| 时间态度 | C/D | Control (掌控时间) vs Deadline (放养) |
| 沟通方式 | V/S | Vocal (话痨) vs Silent (沉默) |
| 决策风格 | R/E | Result (结果导向) vs Emotion (情感导向) |
| 执行偏好 | I/G | Idea (创意型) vs Grunt (执行型) |

## 题目逻辑

### 基础题目 (10题) - 所有人相同

1. 离展示还有3天，组内进度50%
2. 你的part被打回重做3次
3. 小组讨论时的角色
4. 看到组员发"收到"但24小时没动静
5. 分配任务时的倾向
6. 面对分歧时
7. 当有人说"这个我不会"
8. 小组作业截止前24小时
9. 完成作业后
10. 收到群里99+消息

### 角色专属题目 (5题)

**组长题目:**
1. 刚被选为组长
2. 组员说"不想做"
3. 看到组员进度慢
4. 展示前一周
5. 组员做出了成果

**组员题目:**
1. 被分配到不想做的部分
2. 组长催进度时
3. 做作业时喜欢的方式
4. 组长提出新想法
5. 组员摆烂

## 详细人格描述

### CVRI - 掌控话痨结果创意型
**组长:** "我全都要！" 全部都要掌控在自己手里
- 你是组内最强势的组长，所有事情都要按你的来。沟通要频繁，结果要完美，还要有创意。组员怕你，但成果确实牛。
- 名场面: 建群、制定计划、疯狂催促、追求完美、还要创新

**组员:** "我可以做得更好！"
- 最卷的组员，不仅按组长说的做，还要更好。沟通频繁，追求完美，还常有新想法。

### CVRG - 掌控话痨结果执行型
**组长:** "做就完事！"
- 行动派组长：掌控全局、频繁沟通、只看结果、埋头执行

**组员:** "做就对了！"
- 最靠谱的组员：按组长说的做，执行到位，经常提前完成

### CVEI - 掌控话痨情感创意型
**组长:** "大家开心最重要！"
- 最有人情味的组长：掌控进度、频繁沟通、照顾情绪、创意不断

**组员:** "大家开心我也开心！"
- 组内的开心果：话多、情绪价值拉满

### CVEG - 掌控话痨情感执行型
**组长:** "我来���，你们开心就好"
- 最操心的组长：掌控节奏、疯狂催促、照顾情绪、自己兜底

**组员:** "我支持你！"
- 最暖的组员：配合执行还给情绪价值

### CSRI - 掌控沉默结果创意型
**组长:** "成果说话"
- 低调的强者：掌控进度、沉默但关注结果、时有创意

**组员:** "低调发育"
- 最卷的潜水员：不多说话，但偷偷努力拿结果

### CSRG - 掌控沉默结果执行型
**组长:** "干就对了！"
- 闷声干大事的组长：掌控进度、不多废话、只问结果

**组员:** "好的，我做"
- 最靠谱的执行者：不多废话，交代的事一定做好

### CSEI - 掌控沉默情感创意型
**组长:** "大家开心我就开心"
- 温暖的掌控者：掌控进度、不多说话、照顾情绪

**组员:** "加油！你可以的！"
- 温暖的小太阳：话不多但给足情绪支持

### CSEG - 掌控沉默情感执行型
**组长:** "我来做，大家开心"
- 暖心大管家：掌控一切但不多说，照顾大家情绪

**组员:** "好呀，可以呀"
- 最配合的组员：让做什么做什么，情绪价值给满

### DVRI - 放养话痨结果创意型
**组长:** "我有个大胆的想法"
- 最自由的组长：不管进度、疯狂输出、只看结果

**组员:** "我有超棒的想法！"
- 创意达人：DDL前才动工常有神来之笔

### DVRG - 放养话痨执行型
**组长:** "加油！奥利给！"
- 最佛的话痨组长：不管进度、光说不练

**组员:** "加油！你可以的！"
- 气氛组组长：话多、情绪价值给满

### DVEI - 放养话痨情感创意型
**组长:** "大家开心最重要！"
- 温暖的放养组长：不管进度、话很多、情绪价值拉满

**组员:** "开心就好啦！"
- 组内的开心果：DDL前才动工、话多

### DVEG - 放养话痨情感执行型
**组长:** "加油就完事！"
- 最佛系的组长：话多、情绪价值拉满、但自己不动手

**组员:** "加油！你一定行！"
- 气氛组：话多、加油多、但实际行动不多

### DSRI - 放养沉默结果创意型
**组长:** "结果说话"
- 最神秘的组长：不管进度、沉默少言、只看结果

**组员:** "做就对了"
- 最靠谱的潜水员：话少但靠谱

### DSRG - 放养沉默结果执行型
**组长:** "勿cue"
- 发完任务就消失，只看结果不管过程

**组员:** "做就完了"
- 最靠谱的执行者：话少活多

### DSEI - 放养沉默情感创意型
**组长:** "大家开心就好"
- 最温暖的神秘组长：偶尔冒出温暖的话

**组员:** "加油~"
- 温暖但低调的存在：话不多但很暖心

### DSEG - 放养沉默情感执行型
**组长:** "我都可以"
- 最佛系的组长：什么都行、随便你们

**组员:** "我都行"
- 最透明的存在：让做什么做什么，配合度高

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端，单文件部署
- 响应式设计，支持移动端

## 本地运行

```bash

open index.html

# 方式2: HTTP服务器

```
